# Contributing to SagaTheHorde

Thank you for your interest in contributing to the SagaTheHorde DayZ mod! This document provides guidelines for contributing to the project.

## How to Contribute

### Reporting Bugs
If you find a bug, please open an issue with:
- Clear description of the problem
- Steps to reproduce
- Expected behavior vs actual behavior
- Server configuration (player count, map, other mods)
- Any error messages from server logs

### Suggesting Features
Feature suggestions are welcome! Please open an issue with:
- Clear description of the feature
- Use case and benefits
- How it would fit with existing functionality
- Any implementation ideas you have

### Code Contributions

#### Before You Start
1. Check existing issues to see if someone is already working on it
2. For major changes, open an issue first to discuss
3. Fork the repository
4. Create a branch for your changes

#### Development Guidelines

##### Code Style
- Follow existing code formatting and style
- Use clear, descriptive variable and function names
- Add comments for complex logic
- Keep functions focused and manageable in size

##### Testing
- Test your changes on a DayZ server
- Test with different player counts
- Test on different maps if possible
- Document any edge cases you discover

##### Documentation
- Update README.md if user-facing features change
- Update INSTALL.md if installation steps change
- Update CONFIGURATION.md if new configuration options added
- Add entries to CHANGELOG.md for your changes

#### Pull Request Process
1. Ensure your code works and doesn't break existing functionality
2. Update documentation as needed
3. Commit with clear, descriptive messages
4. Create a pull request with:
   - Description of changes
   - Why the changes are needed
   - Any testing performed
   - Screenshots/videos if applicable

### Areas for Contribution

We welcome contributions in these areas:

#### Code Enhancements
- Performance optimizations
- New zombie types or behaviors
- Difficulty scaling systems
- Admin commands
- Configuration file support
- Statistics and logging
- Integration with other mods

#### Documentation
- Improve installation instructions
- Add more configuration examples
- Create video tutorials
- Translate documentation to other languages
- Add troubleshooting tips

#### Testing
- Test on different server configurations
- Test with various player counts
- Test on different maps
- Report compatibility issues
- Performance testing

#### Design
- Create mod logo/icon
- Design promotional materials
- Improve documentation formatting

## Code of Conduct

### Our Standards
- Be respectful and inclusive
- Welcome newcomers and help them learn
- Focus on constructive criticism
- Accept feedback gracefully
- Put community interests first

### Unacceptable Behavior
- Harassment or discrimination
- Trolling or insulting comments
- Personal or political attacks
- Publishing others' private information
- Other unprofessional conduct

## Questions?

If you have questions about contributing:
- Check existing documentation
- Look through closed issues for similar questions
- Open a new issue with your question
- Be patient and respectful

## Recognition

Contributors will be:
- Listed in CHANGELOG.md for their contributions
- Credited in release notes
- Appreciated by the community!

Thank you for helping make SagaTheHorde better!
